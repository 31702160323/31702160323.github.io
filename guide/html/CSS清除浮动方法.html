# CSS 清除浮动方法

  1. 父级节点设置高度

    **优点：**简单、代码少；

    **缺点：**只适合高度固定的布局，要给出精确的高度、如果高度和父类节点不一样是，会产生问题；

  2. 添加一个空 div，利用 css 提高的 clear:both 清除浮动，让父级 div 能自动获取到高度

    **优点：**clear:both
