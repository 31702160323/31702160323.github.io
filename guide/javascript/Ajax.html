# Ajax

# 简介

Asynchronous JavaScript + XML（异步JavaScript和XML）, 其本身不是一种新技术，而是一个在 2005年被Jesse James Garrett提出的新术语，用来描述一种使用现有技术集合的‘新’方法，包括: HTML (opens new window) 或 XHTML (opens new window), CSS (opens new window), JavaScript (opens new window), DOM (opens new window), XML (opens new window), XSLT (opens new window), 以及最重要的 XMLHttpRequest (opens new window)。当使用结合了这些技术的AJAX模型以后， 网页应用能够快速地将增量更新呈现在用户界面上，而不需要重载（刷新）整个页面。这使得程序能够更快地回应用户的操作。

尽管X在Ajax中代表XML, 但由于JSON (opens new window)的许多优势，比如更加轻量以及作为Javascript的一部分，目前JSON的使用比XML更加普遍。JSON和XML都被用于在Ajax模型中打包信息。

  1. 创建XMLHttpRequest对象

    // 在IE浏览器中创建XMLHttpRequest对象的方式为:
var xmlHttpRequest = new ActiveXObject("Microsoft.XMLHTTP");

// 在Netscape浏览器中创建XMLHttpRequest对象的方式为:
var xmlHttpRequest = new XMLHttpRequest();

  2. 创建HTTP请求

    XMLHttpRequest.open(method,URL,flag,name,password);

    参数说明:

    • **method：**该参数用于指定HTTP的请求方法，一共有get、post、head、put、delete五种方法，常用的方法为get和post。
    • **URL：**该参数用于指定HTTP请求的URL地址，可以是绝对URL，也可以是相对URL。
    • **flag：**该参数为可选，参数值为布尔型。该参数用于指定是否使用异步方式。true表示异步、false表示同步，默认为true。
    • **name：**该参数为可选参数，用于输入用户名。如果服务器需要验证，则必须使用该参数。
    • **password：**该参数为可选，用于输入密码。若服务器需要验证，则必须使用该参数。

  3. 设置响应HTTP请求状态变化的函数

    xmlHttpRequest.onreadystatechange = function() {};

/** 
1、未初始化状态: 在创建完XMLHttpRequest对象时，此时XMLHttpRequest.readyState为0。
2、初始化状态: 在创建完XMLHttpRequest对象后使用open()方法创建了HTTP请求时。此时XMLHttpRequest.readyState为1。
3、发送数据状态: 在初始化XMLHttpRequest对象后，使用send()方法发送数据时，此时XMLHttpRequest.readyState为2。
4、接收数据状态: Web服务器接收完数据并进行处理完毕之后，向客户端传送返回的结果。此时XMLHttpRequest.readyState为3。
5、完成状态: XMLHttpRequest对象接收数据完毕后，进入完成状态，此时XMLHttpRequest.readyState为4。此时可以使用此时XMLHttpRequest.responseText属性或此时XMLHttpRequest.responseXml属性来获取数据。
*/

  4. 设置获取服务器返回数据的语

    if(xmlHttpRequst.status == 200) {
    document.write(xmlHttpRequest.responseText);//将返回结果以字符串形式输出
    //document.write(xmlHttpRequest.responseXML);//或者将返回结果以XML形式输出
}

  5. 发送HTTP请求

    XMLHttpRequest.send(data)

# 实现AJAX的基本案例

let xmlHttpRequest;  // 第一步: 创建XMLHttpRequest对象，即一个用于保存异步调用对象的变量
if(window.ActiveXObject){// IE浏览器的创建方式
    xmlHttpRequest = new ActiveXObject("Microsoft.XMLHTTP");
}else if(window.XMLHttpRequest){
    xmlHttpRequest = new XMLHttpRequest();
}
xmlHttpRequest.open("GET", url, true);// 第二步: 创建一个新的HTTP请求,并指定该HTTP请求的方法、URL及验证信息.
xmlHttpRequest.onreadystatechange= function(){// 第三步: 设置响应http请求状态变化的事件
    if(xmlHttpRequest.readyState == 4){// 第五步: 获取异步调用返回的数据 
        if(xmlHttpRequest.status == 200) {
            xmlHttpRequest.responseText; // 局部刷新数据到页面
        }
    }
}
xmlHttpRequest.send(null); // 第四步: 发送请求
